# Finance Mate App  

A modern mobile app UI built with Flutter to manage personal finances easily and beautifully.  
This project was developed as part of a Mobile Programming course assignment.  

---

## Features  
 Splash screen with gradient background  
 Interactive home page with balance cards  
 Category grid (Health, Travel, Food, Event)  
 Transaction history list  
 Simple and responsive UI design
